Mail Extended

This module is developed to extend the functionality of mail module. Just like popular mail service gmail and yahoo mail, module gives you option to forward the email, you can directly forward the mails from odoo. It will add a forward button in the mail wall and user will be able to forward an email to recipients. Forward button will be displayed as shown below.

When user clicks on forward button, it will display a new popup with the message body and list of recipents. User can add a recipients and message will be forwarded to a list of recipients.

On an another side, recipients will see the message as displayed above.
